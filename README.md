# Tarefa de TypeScript – INJunior (2026.1)

Resolução da **tarefa de TypeScript** proposta no módulo introdutório da INJunior – 2026.1.

O projeto contém a implementação dos exercícios solicitados, com foco em **tipagem estática**, **boas práticas**, **organização de código** e **fidelidade ao enunciado**.

---

## Tecnologias

- TypeScript
- Node.js
- NPM

---

## Estrutura do Projeto

exercicios/
├── exercicio1.ts # Variáveis e inferência de tipos
├── exercicio2.ts # Type Alias e Union Types
├── exercicio3.ts # Interface e função de resumo
├── exercicio4.ts # Interface com método

├── exercicio6.ts # Função genérica
└── exercicio7.ts # Interface genérica
